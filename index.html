Коротко про вебдизайн

HTML

HTML (англ. HyperText Markup Language — мова розмітки гіпертексту) — стандартизована мова розмітки документів для перегляду вебсторінок у браузері. Браузери отримують HTML-документ від сервера за протоколами HTTP/HTTPS або відкривають з локального диска, далі інтерпретують код в інтерфейс, який відображається на екрані. Елементи HTML є будівельними блоками сторінок. HTML позначає структурну семантику тексту, наприклад заголовки, абзаци, списки, посилання, цитати та інше.

Детальніше

CSS

CSS (англ. Cascading Style Sheets, укр. Каскадні таблиці стилів) — мова стилів, що використовується для опису вигляду вебсторінок. CSS є однією з основних технологій веба поряд із HTML та JavaScript. Вона дозволяє візуально форматувати елементи сторінки, змінювати кольори, шрифти, розташування та інші властивості.

Детальніше

JavaScript

JavaScript (JS) — динамічна, об'єктно-орієнтована мова програмування. Використовується для написання сценаріїв вебсторінок, що дозволяє взаємодіяти з користувачем, надсилати/отримувати дані з сервера без перезавантаження сторінки.

Детальніше

Теги

html
Визначає початок та кінець HTML-документа.
head
Визначає метадані вебсторінки, такі як заголовок, ключові слова тощо.
title
Встановлює заголовок сторінки, який показується у вкладці браузера.
body
Визначає основний вміст вебсторінки, який бачить користувач.
